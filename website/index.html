强大功能，得心应手

精心设计的每一个细节，让 Markdown 创作更加高效

项目文件树

以熟悉的文件树结构组织管理你的 Markdown 文档，支持展开折叠、多标签编辑，侧栏宽度可调。

实时预览

内置 CodeMirror 6 编辑器与 Marked 渲染引擎，编辑与预览无缝切换，支持行号显示。

拖拽排序

使用 @dnd-kit 实现直观的拖拽操作，轻松调整文件与文件夹的顺序和层级结构。

文件操作

支持新建、重命名、删除、复制/粘贴文件与文件夹，一切操作都在主进程中安全执行。

元数据管理

为节点设置备注说明、颜色标记等元数据，方便分类和查找你的文档。

外部打开

使用系统默认应用或指定应用打开文件，与其他工具无缝协作。

Electron 35 React 19 TypeScript 5.8 CodeMirror 6 Webpack 5 Sass

主进程

Node.js 环境负责目录扫描、文件监听与磁盘操作，确保安全高效的文件管理。

渲染进程

React 19 + Webpack Dev Server 实现热更新开发，实时预览编辑效果。

编辑器

CodeMirror 6 提供专业级 Markdown 编辑体验，支持语法高亮与多种编辑功能。

快速开始

只需几步，即可开启你的 Markdown 项目管理之旅

1

克隆项目

git clone https://github.com/Reagan1947/Typora-Explore.git
2

安装依赖

cd Typora-Explore && npm install
3

启动开发

npm start
4

打包应用

npm run package
