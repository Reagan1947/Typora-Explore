# 文件树需求



## 1.1 自由排序与标记和备注



### 1.1.1 自由排序



用户可对文件树中的内容通过拖拽的方式进行自由排序, 文件夹内文件的排序不需要同步到系统磁盘。文件夹之间的排序也不需要同步到磁盘。而是通过固化到持久化文件中实现。



1. 但是如果涉及文件夹之间文件的顺序调整，例如用户将文件从一个文件夹移动到另一个文件夹。这种情况需要同步到磁盘中
2. 另外如果用户将一个文件夹变为另一个文件夹的子文件夹也需要同步到磁盘。











### 1.1.2 标记和备注



用户右键点击文件时在可以通过右键菜单中的 Edit Detail 功能编辑文件的备注信息和标记。备注信息展示在文件名后，标记信息则展示在文件正行最末尾处。文件夹同样支持标记和备注。所有信息编辑后保存到持久化文件中



## 1.2 复制粘贴、重命名、删除



### 1.2.1 复制粘贴



用户可复制文件并进行粘贴操作，粘贴时不需要检查文件名是否重复。可跨文件夹复制粘贴。复制粘贴结果需要同步到系统磁盘中。并更新持久化文件



### 1.2.2 重命名



用户可通过右键菜单进行文件和文件夹重命名操作，文件重命名后也需要同步到系统磁盘。并更新持久化文件



### 1.2.3 删除



用户可删除文件和文件夹，删除结果需要同步到系统磁盘。并更新持久化文件



## 1.3 文件夹监控以及在文件系统中查看



### 1.3.1 文件夹监控



用户若在系统磁盘文件夹中进行了删除、新增、重命名等操作时需要同步到应用。并更新 持久化文件，注意文件夹内的文件排序动作不需要同步到应用。同级文件夹之间的排序也不需要同步到应用。



### 1.3.2 在文件系统中查看



用户可通过右键菜单，选择在对应文件系统中查看文件。



## 1.4 新增文件或文件夹



### 1.4.1 新增文件或文件夹



用户可以通过顶部工具栏的新建按钮选择新建文件或文件夹，新增的文件夹命名为 Untitled 新增的文件也命名为 Untitled，且需要同步到系统磁盘中，并更新持久化文件
