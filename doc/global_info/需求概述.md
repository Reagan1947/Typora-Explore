# Markdown 文件管理工具 - 需求说明文档

## 项目概述

本项目是一个基于 Web 的 Markdown 文件管理和编辑工具，UI 设计参考 JetBrains IDEA 风格，提供直观的文件树管理和强大的编辑功能。

## 设计风格

### UI 主题

- **设计参考**: JetBrains IDEA
- **配色方案**: 深色主题
- **主要颜色**:
  - 背景色: #2B2D30
  - 边框色: #3C3F41
  - 选中高亮: #4A88C7
  - 悬停背景: #313335
  - 文本色: #BCBEC4

## 核心功能

### 1. 文件树管理

#### 1.1 基础操作

- **文件/文件夹显示**: 层级树形结构展示
- **展开/折叠**: 点击文件夹可展开或折叠子项
- **文件选择**: 点击文件在编辑器中打开
- **视觉反馈**: 
  - 选中状态高亮显示（蓝色背景）
  - 鼠标悬停效果（灰色背景）

#### 1.2 拖放排序

- **技术实现**: 使用 react-dnd 库
- **支持操作**:
  - 文件/文件夹可任意拖动
  - 支持跨层级拖动
  - 拖动时显示放置位置指示器（before/after/inside）
- **拖放规则**:
  - 文件可拖放到文件之前/之后，或文件夹内部
  - 文件夹可拖放到其他文件夹内部，或文件/文件夹之前/之后
  - 不能将文件夹拖入其自身或子文件夹

#### 1.3 新建操作

- **新建文件**: 
  - 自动创建名为 `untitled.md` 的文件
  - 如存在重名，自动添加数字后缀（untitled 2.md, untitled 3.md...）
  - 创建后自动在编辑器中打开
- **新建文件夹**:
  - 自动创建名为 `New Folder` 的文件夹
  - 如存在重名，自动添加数字后缀（New Folder 2, New Folder 3...）
- **创建方式**: 通过工具栏的加号按钮下拉菜单选择

#### 1.4 文件操作

- **重命名**: 
  - 右键菜单选择 "Rename"
  - 直接在文件树中编辑文件名
  - 重名检测：同级目录下不允许重名
- **删除**: 
  - 右键菜单选择 "Delete"
  - 删除文件夹会连同子项一起删除
  - 删除已打开的文件会关闭对应标签页
- **复制粘贴**:
  - 右键菜单选择 "Copy" 复制节点
  - 右键菜单选择 "Paste" 粘贴到目标位置
  - 支持深度克隆（包括所有子节点）
  - 自动生成新 ID
  - 粘贴的文件名自动添加 "copy" 后缀
  - 重名时自动累加数字（file copy.md → file copy 2.md）

#### 1.5 文件编辑详情

- **备注功能**: 为文件/文件夹添加备注文本
- **颜色标记**: 支持 8 种颜色标记
  - 红色、橙色、黄色、绿色、蓝色、紫色、粉色、灰色
- **显示方式**:
  - 备注以斜体显示在文件名后方
  - 颜色标记以小圆点显示在行尾
- **编辑方式**: 右键菜单选择 "Edit Details" 打开编辑对话框

#### 1.6 工具栏功能

- **新建菜单**: 包含新建文件、新建文件夹、搜索功能
- **定位文件**: 在文件树中定位当前打开的文件
- **展开所有**: 展开所有文件夹
- **折叠所有**: 折叠所有文件夹

### 2. Markdown 编辑器

#### 2.1 编辑功能

- **技术实现**: 基于 CodeMirror 6
- **语法高亮**: 完整的 Markdown 语法高亮支持
- **实时编辑**: 内容修改实时保存到内存
- **修改状态**: 
  - 已修改的文件标签页显示圆点标记
  - Ctrl+S 保存文件（模拟保存）

#### 2.2 编辑器设置

- **行号显示**: 
  - 编辑器右上角切换按钮
  - 可显示/隐藏行号
  - 设置保存到 localStorage
- **主题**: 深色主题，匹配整体 UI 风格
- **字体**: 等宽字体，适合代码编辑

### 3. 多标签页管理

#### 3.1 标签页功能

- **自动打开**: 点击文件自动创建标签页
- **标签切换**: 点击标签页切换到对应文件
- **关闭标签**: 点击标签上的 × 按钮关闭
- **修改标记**: 已修改文件的标签显示小圆点
- **视觉效果**:
  - 活动标签页深色背景
  - 非活动标签页浅色背景
  - 悬停时显示高亮效果

### 4. 搜索功能

#### 4.1 文件搜索

- **搜索位置**: 工具栏加号菜单中的搜索框
- **搜索范围**: 搜索文件和文件夹名称
- **实时过滤**: 输入时实时过滤文件树
- **高亮显示**: 
  - 匹配的节点及其父节点显示
  - 自动展开包含匹配项的文件夹
- **清除搜索**: 点击搜索框的 × 按钮清除

#### 4.2 全局搜索

- **打开方式**: 顶部工具栏搜索图标
- **搜索内容**: 搜索文件名和文件内容
- **搜索结果**: 
  - 显示匹配的文件列表
  - 显示匹配的内容片段
  - 点击结果打开对应文件

### 5. 项目管理

#### 5.1 项目列表

- **项目显示**: 启动时显示项目列表
- **项目信息**: 
  - 项目名称
  - 项目路径
  - 创建时间
  - 最后修改时间
- **项目操作**:
  - 打开项目
  - 编辑项目信息
  - 删除项目

#### 5.2 项目创建

- **创建方式**:
  - 新建空白项目
  - 从本地文件夹导入
- **项目设置**:
  - 项目名称
  - 项目路径（可选）
- **数据持久化**: 使用 localStorage 保存项目列表

#### 5.3 本地导入

- **导入功能**: 从本地文件夹导入文件结构
- **文件过滤**: 仅导入 .md 和 .markdown 文件
- **结构保持**: 保持原有的文件夹层级结构
- **内容读取**: 读取 Markdown 文件内容到编辑器

### 6. 设置功能

#### 6.1 编辑器设置

- **行号显示**: 开启/关闭行号显示
- **字体大小**: 调整编辑器字体大小
- **主题设置**: 选择编辑器主题（预留）

#### 6.2 通用设置

- **自动保存**: 自动保存间隔设置
- **语言设置**: 界面语言选择（预留）
- **快捷键**: 自定义快捷键（预留）

## 技术栈

### 前端框架

- **React**: 18.x
- **TypeScript**: 类型安全
- **Tailwind CSS**: V4，样式系统

### 核心库

- **react-dnd**: 拖放功能实现
- **react-dnd-html5-backend**: HTML5 拖放后端
- **@codemirror/view**: CodeMirror 6 视图层
- **@codemirror/state**: CodeMirror 6 状态管理
- **@codemirror/lang-markdown**: Markdown 语法支持
- **@radix-ui/react-dropdown-menu**: 下拉菜单组件
- **@radix-ui/react-dialog**: 对话框组件
- **lucide-react**: 图标库

### 数据持久化

- **localStorage**: 存储项目列表、设置、文件内容

## 交互细节

### 快捷键

- **Ctrl+S**: 保存当前文件
- **Escape**: 
  - 关闭对话框
  - 取消编辑状态
  - 清除搜索

- **Rename**: 重命名文件/文件夹
- **Edit Details**: 编辑备注和颜色标记
- **Delete**: 删除文件/文件夹
- **Copy**: 复制节点
- **Paste**: 粘贴节点（复制后可用）

### 视觉反馈

- **拖放指示器**: 
  - 蓝色线条指示拖放位置（before/after）
  - 蓝色边框指示拖入文件夹（inside）
- **选中状态**: 蓝色背景高亮
- **悬停状态**: 灰色背景提示
- **修改标记**: 标签页小圆点提示未保存
